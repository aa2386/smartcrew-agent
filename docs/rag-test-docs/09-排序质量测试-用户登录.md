# 用户登录流程

## 登录方式

SmartCrew 支持以下登录方式：
- 用户名密码登录
- 飞书扫码登录
- 企业微信登录

## 用户名密码登录

### 请求接口

```
POST /api/v1/auth/login
Content-Type: application/json

{
    "username": "admin",
    "password": "<configured-admin-password>"
}
```

### 响应示例

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "token": "eyJhbGciOiJIUzI1NiIs...",
        "expireIn": 720,
        "user": {
            "id": 1,
            "username": "admin",
            "displayName": "系统管理员"
        }
    }
}
```

## 登录状态管理

登录成功后，Token 存储在客户端，每次请求携带 Token：

```
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
```

## 登出流程

```
POST /api/v1/auth/logout
```

登出后 Token 失效，需要重新登录。

## 登录失败处理

| 错误码 | 说明 |
|--------|------|
| 401 | 用户名或密码错误 |
| 403 | 账户被禁用 |
| 429 | 登录尝试次数过多 |

## 管理员默认账号

系统启动时会自动创建管理员账号：
- 用户名：admin
- 密码：<configured-admin-password>

请在首次登录后立即修改密码。
