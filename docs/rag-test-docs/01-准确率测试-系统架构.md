# SmartCrew 系统架构概述

## 系统简介

SmartCrew 是一个基于 LangChain4j 框架构建的智能 Agent 平台，采用 Spring Boot 3.x 作为基础框架，支持多模态交互和工具调用能力。

## 核心模块

### 1. smartcrew-admin

管理端入口模块，提供 Web 服务和 REST API 接口。主要职责包括：
- 启动 Spring Boot 应用
- 配置 Web MVC 和安全策略
- 提供 Controller 层接口

### 2. smartcrew-modules

核心业务模块，包含以下子模块：
- **Agent 模块**：InitialAgent、StubAgent 等智能体实现
- **LLM 模块**：大模型客户端封装，支持 DashScope
- **Memory 模块**：会话记忆和用户偏好管理
- **Tool 模块**：工具注册和执行
- **Platform 模块**：飞书、企业微信等平台适配

### 3. smartcrew-modules-api

API 定义模块，定义服务接口和领域对象：
- `api/agent`：Agent 服务接口
- `api/llm`：LLM 服务接口
- `api/memory`：记忆服务接口
- `api/tool`：工具服务接口

### 4. smartcrew-common

公共模块，包含通用工具和配置：
- 认证鉴权
- 异常处理
- 通用工具类

## 技术栈

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.4.4 | 基础框架 |

## 配置说明

### 应用配置

```yaml
server:
  port: 8085

smartcrew:
  name: smartcrew-agent
  version: 0.0.1-SNAPSHOT
  llm:
    enabled: true
    provider: dashscope
    model: qwen-plus
```

### 数据库配置

系统支持 MySQL 和 H2 数据库，开发环境默认使用 H2 内存数据库。

## 启动方式

```bash
# 编译项目
mvn clean package -DskipTests

# 启动服务
java -jar smartcrew-admin/target/smartcrew-admin-0.0.1-SNAPSHOT.jar
```

服务启动后访问：http://localhost:8085
