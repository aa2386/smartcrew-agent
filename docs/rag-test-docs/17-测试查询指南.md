## 评估指标说明

### 计算公式

- **准确率** = 正确返回的文档数 / 返回的文档总数
- **召回率** = 正确返回的文档数 / 知识库中相关的文档总数
- **MRR** = 第一个相关文档排名的倒数之和 / 查询总数
- **F1** = 2 × 准确率 × 召回率 / (准确率 + 召回率)

### 评分标准

| 指标 | 优秀 | 良好 | 合格 | 不合格 |
|------|------|------|------|--------|
| 准确率 | >90% | >80% | >70% | <70% |
| 召回率 | >90% | >80% | >70% | <70% |
| MRR | >0.9 | >0.8 | >0.7 | <0.7 |

4. **注意检查相似度阈值设置**

## 常见问题

### 查询返回空结果

- 检查 Chroma 服务是否正常运行
- 确认文档已成功索引
- 验证嵌入模型配置正确

### 返回结果不相关

- 调整分割参数
- 检查嵌入模型维度是否匹配
- 尝试使用其他嵌入模型
