# RAG 与知识图谱结合

## 概述

将 RAG 与知识图谱结合，可以提升检索的语义理解能力和推理能力。

## 结合方式

### 1. 知识图谱增强检索

- 利用实体关系扩展查询
- 通过图谱路径发现隐含关联
- 提高检索的语义理解

### 2. 混合检索策略

```
用户查询 → 向量检索 + 图谱检索 → 结果融合 → 生成回答
```

### 3. 知识图谱构建

从文档中抽取实体和关系：
- 实体识别（NER）
- 关系抽取（RE）
- 知识融合

## 优势

| 优势 | 说明 |
|------|------|
| 语义增强 | 理解实体间关系 |
| 推理能力 | 支持多跳推理 |
| 可解释性 | 知识来源可追溯 |

## 挑战

| 挑战 | 说明 |
|------|------|
| 构建成本 | 需要大量标注数据 |
| 维护复杂 | 知识更新需要同步 |
| 技术门槛 | 需要图谱专业知识 |

## SmartCrew 规划

SmartCrew 计划在未来版本中支持：
- 实体识别和抽取
- 知识图谱存储
- 图谱增强检索
