## 六、评估指标计算

### 6.1 准确率（Precision）

**定义：** 返回结果中相关文档的比例

**计算公式：**
```
准确率 = 正确返回的文档数 / 返回的文档总数
```

**示例：**
- 返回 3 条结果，其中 2 条相关
- 准确率 = 2/3 = 66.67%

**评分标准：**

| 等级 | 准确率范围 |
|------|------------|
| 优秀 | ≥ 90% |
| 良好 | 80% - 89% |
| 合格 | 70% - 79% |
| 不合格 | < 70% |

### 6.2 召回率（Recall）

**定义：** 相关文档被返回的比例

**计算公式：**
```
召回率 = 正确返回的文档数 / 知识库中相关的文档总数
```

**示例：**
- 知识库中有 4 篇相关文档
- 返回结果中包含 3 篇
- 召回率 = 3/4 = 75%

**评分标准：**

| 等级 | 召回率范围 |
|------|------------|
| 优秀 | ≥ 90% |
| 良好 | 80% - 89% |
| 合格 | 70% - 79% |
| 不合格 | < 70% |

### 6.3 F1 分数

**定义：** 准确率和召回率的调和平均

**计算公式：**
```
F1 = 2 × (Precision × Recall) / (Precision + Recall)
```

**示例：**
- 准确率 = 80%，召回率 = 75%
- F1 = 2 × (0.8 × 0.75) / (0.8 + 0.75) = 77.42%

### 6.4 MRR（Mean Reciprocal Rank）

**定义：** 第一个相关文档排名的倒数平均值

**计算公式：**
```
MRR = (1/N) × Σ(1/rank_i)
```

**示例：**
- 查询1：第一个相关文档排在第 1 位 → 1/1 = 1.0
- 查询2：第一个相关文档排在第 2 位 → 1/2 = 0.5
- 查询3：第一个相关文档排在第 1 位 → 1/1 = 1.0
- MRR = (1.0 + 0.5 + 1.0) / 3 = 0.83

**评分标准：**

| 等级 | MRR 范围 |
|------|----------|
| 优秀 | ≥ 0.9 |
| 良好 | 0.8 - 0.89 |
| 合格 | 0.7 - 0.79 |
| 不合格 | < 0.7 |

### 6.5 NDCG（Normalized Discounted Cumulative Gain）

**定义：** 考虑排序位置的评估指标

**计算公式：**
```
DCG = Σ(rel_i / log2(i+1))
NDCG = DCG / IDCG
```
