# RAG 常见问题排查

## 检索问题

### 问题：检索结果不相关

可能原因：
1. 嵌入模型不适合中文场景
2. 文档分割粒度不当
3. 查询与文档语义差距大

解决方案：
- 选择适合的嵌入模型（如 text-embedding-v3）
- 调整分割参数（chunk_size, overlap）
- 使用查询改写技术

### 问题：召回率低

可能原因：
1. 文档覆盖不全
2. 检索 top-k 设置过小
3. 相似度阈值过高

解决方案：
- 补充相关文档
- 增加 top-k 值
- 降低 min_score 阈值

## 生成问题

### 问题：回答包含幻觉

可能原因：
1. 检索内容不足
2. LLM 过度发挥
3. 提示词设计不当

解决方案：
- 提高检索质量
- 在提示词中强调"基于参考资料回答"
- 设置温度参数为较低值

### 问题：回答格式不统一

解决方案：
- 在提示词中指定输出格式
- 使用结构化输出

## 系统问题

### 问题：响应速度慢

可能原因：
1. 向量数据库查询慢
2. 嵌入计算耗时
3. LLM 生成慢

解决方案：
- 优化向量索引
- 使用批量嵌入
- 考虑使用更快的模型
