# 工具系统设计

## 工具概述

SmartCrew 的工具系统允许 Agent 调用外部功能，如文件操作、网页搜索、图片生成等。

## 工具定义

### 工具接口

工具通过 Spring Bean 的方式定义，使用 `@Tool` 注解标记方法：

```java
@Component
public class BasicTools {

    @Tool("获取当前时间")
    public String getCurrentTime() {
        return LocalDateTime.now().format(DateTimeFormatter.ISO_LOCAL_DATE_TIME);
    }

    @Tool("计算两个数的和")
    public int add(@P("第一个数") int a, @P("第二个数") int b) {
        return a + b;
    }
}
```

### 工具注册

工具定义存储在 `tool_definition` 表中：

```sql
CREATE TABLE tool_definition (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    tool_code VARCHAR(64) NOT NULL,
    tool_name VARCHAR(128) NOT NULL,
    description VARCHAR(255) NOT NULL,
    bean_name VARCHAR(128) NOT NULL,
    risk_level VARCHAR(32) NOT NULL,
    enabled TINYINT(1) NOT NULL DEFAULT 1
);
```

## 内置工具

### BasicTools

基础工具集，包含：
- `getCurrentTime()`：获取当前时间
- `add(a, b)`：计算两数之和

### FileTools

文件操作工具：
- `readFile(path)`：读取文件内容
- `writeFile(path, content)`：写入文件
- `listFiles(directory)`：列出目录文件

### WebSearchTools

网页搜索工具：
- `search(query)`：搜索网页
- `fetchContent(url)`：获取网页内容

### ImageSearchTools

图片搜索工具：
- `searchImage(query)`：搜索图片

### PlantUmlTools

PlantUML 图表生成：
- `generateDiagram(umlCode)`：生成 UML 图

## 工具绑定

Agent 通过 `agent_tool_binding` 表绑定工具：

```sql
INSERT INTO agent_tool_binding (agent_code, tool_code)
VALUES ('initial-agent', 'basic-tools');
```

## 工具执行流程

```
用户请求 → Agent 判断是否需要工具 → LLM 生成工具调用 → 执行工具 → 返回结果
```

## 风险等级

| 等级 | 说明 | 示例 |
|------|------|------|
| LOW | 低风险，只读操作 | 获取时间、搜索 |
| MEDIUM | 中风险，有限写入 | 写入临时文件 |
| HIGH | 高风险，系统操作 | 执行命令、删除文件 |

## 配置开关

在 `application.yml` 中配置工具开关：

```yaml
smartcrew:
  tools:
    enabled:
      basic: true
      file: true
      image-search: true
      plantuml: true
      web-search: true
      terminal: false
```
