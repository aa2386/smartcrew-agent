# 用户偏好设置

## 概述

用户偏好设置允许用户自定义系统行为和显示方式。

## 偏好类型

| 类型 | 说明 | 示例 |
|------|------|------|
| theme | 主题设置 | dark, light |
| language | 语言设置 | zh-CN, en-US |
| notification | 通知设置 | enabled, disabled |

## 存储结构

```sql
CREATE TABLE user_preference (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    pref_key VARCHAR(64) NOT NULL,
    pref_value TEXT NOT NULL,
    pref_type VARCHAR(32) NOT NULL,
    source VARCHAR(32) NOT NULL
);
```

## API 接口

### 获取偏好

```
GET /api/v1/preferences/{key}
```

### 设置偏好

```
POST /api/v1/preferences
{
    "key": "theme",
    "value": "dark",
    "type": "string"
}
```

## 偏好继承

用户偏好可以继承自：
- 系统默认值
- 部门设置
- 用户自定义

优先级：用户自定义 > 部门设置 > 系统默认
