# 数据库配置指南 - H2

## 概述

SmartCrew 支持使用 H2 内存数据库，适合开发和测试环境。

## 依赖配置

在 `pom.xml` 中添加 H2 依赖：

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.3.232</version>
</dependency>
```

## 连接配置

在 `application.yml` 中配置数据库连接：

```yaml
spring:
  datasource:
    driver-class-name: org.h2.Driver
    url: jdbc:h2:mem:smartcrew;DB_CLOSE_DELAY=-1;MODE=MySQL
    username: sa
    password: 
  h2:
    console:
      enabled: true
      path: /h2-console
```

## 持久化配置

如需持久化数据到文件：

```yaml
spring:
  datasource:
    url: jdbc:h2:file:./data/smartcrew;MODE=MySQL
```

## H2 控制台

启动应用后访问 H2 控制台：
- URL: http://localhost:8085/h2-console
- JDBC URL: jdbc:h2:mem:smartcrew
- User: sa
- Password: (空)

## 初始化脚本

配置自动执行初始化脚本：

```yaml
spring:
  sql:
    init:
      mode: always
      schema-locations: classpath:sql/init-smartcrew-agent.sql
```

## 注意事项

1. H2 内存数据库重启后数据会丢失
2. 生产环境建议使用 MySQL
3. H2 兼容 MySQL 语法，但部分功能有限制
